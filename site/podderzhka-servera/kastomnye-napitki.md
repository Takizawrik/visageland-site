# Кастомные напитки

## Кастомный рецепт напитка 🍷

На сервере вы можете создать **собственный уникальный напиток** с индивидуальными характеристиками! Это отличный способ проявить креативность и оставить свой след в игровом мире. Однако каждый рецепт проходит проверку на баланс, чтобы сохранить честную и увлекательную игру для всех.

***

### Как это работает?

1. Вы придумываете рецепт напитка, заполняя все обязательные пункты (название, эффекты, ингредиенты и т.д.).
2. Заявка отправляется на рассмотрение администрации.
3. Рецепт проверяется на:
   * **Соответствие балансу** — чтобы не нарушать игровой процесс.
   * **Техническую реализуемость** — в рамках механик сервера.
4. Если рецепт одобрен, он **появляется на сервере**, но **другие игроки не уведомляются** об этом — им предстоит открыть его самостоятельно!

{% hint style="info" %}
Новые рецепты делают игру разнообразнее и интереснее для всего комьюнити — ваш вклад действительно важен!
{% endhint %}

***

### 🚫 Важные ограничения

* Напиток **не должен нарушать правила сервера** (оскорбительные названия, провокационные описания и т.п.).
* **Максимальный уровень эффектов** ограничен ванильными значениями Minecraft (то есть не выше того, что можно получить в обычной игре).

***

### 📝 Что нужно указать в заявке?

#### Обязательные пункты

| Пункт                                  | Описание                                                                                               |
| -------------------------------------- | ------------------------------------------------------------------------------------------------------ |
| **Название**                           | От 4 до 40 символов.                                                                                   |
| **Описание**                           | Умещается в 2 короткие строчки (текст, который будет отображаться у напитка).                          |
| **Цвет**                               | Можно указать HEX-код (например, `#FF5733`).                                                           |
| **Ингредиенты и их количество**        | Указывайте точные названия предметов (например, `3 яблока`, `1 ведро воды`) — без разночтений.         |
| **Время варки в котле**                | В минутах.                                                                                             |
| **Эффекты, их уровень и длительность** | Перечислите все эффекты, которые даёт напиток, с указанием уровня (I, II и т.д.) и времени в секундах. |

#### Дополнительные (необязательные) пункты

| Пункт                       | Примечание                                                                                                 |
| --------------------------- | ---------------------------------------------------------------------------------------------------------- |
| **Этап дистилляции**        | Количество кругов (если планируется).                                                                      |
| **Время выдержки в бочке**  | 1 год = 20 минут игрового времени.                                                                         |
| **Тип древесины для бочки** | Из какой древесины должна быть изготовлена бочка.                                                          |
| **Градус напитка**          | От -100 до 100. Отрицательное значение понижает градус игрового персонажа (например, делает его "пьяным"). |

### Как подать заявку?

Для подачи заявки на создание кастомного напитка заполните форму по шаблону (все обязательные пункты) и отправьте её администрации через Discord (раздел «Тикеты» или личное сообщение). После рассмотрения вы получите уведомление о решении.

{% hint style="warning" %}
Администрация оставляет за собой право отклонить рецепт без объяснения причин, если он нарушает баланс или правила. Будьте внимательны к деталям!
{% endhint %}
