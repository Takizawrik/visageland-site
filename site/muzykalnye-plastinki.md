# Музыкальные Пластинки

* \
  **Запись музыки** Для записи вашей музыки на пластинку необходимо использовать команду: `/disc` Дальше, записаной пластинкой нужно тыкнуть по проигрывателю

`/disc burn ссылка название` – Создайте диск с пользовательским аудио по URL-адресу. Вам нужно подержать в руке музыкальный диск.

`/disc erase` – Очистить диск от музыки.
