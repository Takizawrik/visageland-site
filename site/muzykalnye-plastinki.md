# Музыкальные Пластинки

### 🎵 Запись музыки на пластинку

Аддон `pv-addon-discs` позволяет создавать собственные музыкальные диски с аудио из интернета.

#### Как это работает:

1. Возьмите в руку **пустой музыкальный диск**.
2. Введите команду:

**/disc burn <ссылка> <название>**&#x20;

Где:

* `<ссылка>` — прямая ссылка на аудиофайл (MP3, OGG, AAC) или ссылка на YouTube/Twitch (поддерживается стриминг).
* `<название>` — имя, которое будет отображаться у пластинки.

3. Дождитесь завершения записи. Диск будет заполнен вашим аудио.
4. Установите записанную пластинку в **музыкальный проигрыватель** (нот-блок или другой совместимый блок), чтобы прослушать трек.

#### Дополнительные команды:

| Команда       | Описание                                                                  |
| ------------- | ------------------------------------------------------------------------- |
| `/disc erase` | Очищает текущий диск от записанной музыки (диск снова становится пустым). |

{% hint style="info" %}
Поддерживаются прямые ссылки на аудиофайлы, а также стриминг с YouTube и SoundCloud.
{% endhint %}
