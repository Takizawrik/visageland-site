# Модификации

## Моды для сервера

Чтобы играть на сервере с максимальным комфортом и использовать все его возможности, мы рекомендуем установить определённые моды. Ниже они разделены на две категории: **обязательные** (без них вы не сможете полноценно взаимодействовать с другими игроками) и **рекомендуемые** (они улучшают производительность, запись видео и визуальный опыт).

***

### Как установить моды?

Чтобы установить моды на наш сервер, выполните следующие шаги:

1. **Откройте папку `.minecraft`**.\
   Самый простой способ:
   * Нажмите комбинацию клавиш **Win + R** (откроется окно «Выполнить»).
   * Введите `%AppData%` и нажмите «ОК».
   * В открывшейся папке найдите папку `.minecraft` — это и есть корневая папка игры.
2. **Найдите папку `mods`**.\
   Внутри `.minecraft` должна быть папка `mods`. Если её нет — создайте её вручную (название строго `mods`).
3. **Скопируйте файлы модов**.\
   Скачанные файлы модов (`.jar`) переместите или скопируйте в папку `mods`.
4. **Запустите игру** — моды автоматически загрузятся.

### ✅ Обязательные моды

Эти моды необходимы для базового общения и работы других модификаций.

| Мод             | Что даёт                                                                                                                           |
| --------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| **PlasmoVoice** | Голосовой чат прямо в игре. С ним вы сможете говорить с другими игроками в реальном времени, как в скайпе или дискорде.            |
| **EmoteCraft**  | Эмоции и анимации. Позволяет вашему персонажу показывать жесты, танцевать, зевать и выражать другие эмоции — это оживляет общение. |
| **Fabric API**  | Базовый мод-ядро для Fabric. Без него ни один другой мод из этого списка не заработает. Устанавливается в первую очередь.          |

***

### ⭐ Рекомендуемые моды

Эти моды не обязательны, но существенно улучшают игровой опыт.

| Мод                       | Что даёт                                                                                                                                                                                  |
| ------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Sodium**                | Мощный оптимизатор производительности. Заменяет Optifine на Fabric, поднимает FPS и добавляет гибкие настройки графики. Имеет множество дополнений.                                       |
| **Indium**                | Дополнение к Sodium, обеспечивающее совместимость с некоторыми другими модами. Рекомендуется ставить вместе с Sodium для стабильности.                                                    |
| **Replay Mod**            | Позволяет записывать игровой процесс и потом просматривать его с разных ракурсов. Полезно и для творчества (создание видео), и для фиксации нарушений (доказательства для администрации). |
| **Remove Loading Screen** | Убирает надоедливый загрузочный экран, который появляется при загрузке ресурспаков. Просто делает игру чище и быстрее.                                                                    |

***
