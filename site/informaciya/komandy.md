# Команды

## Команды сервера

### Регистрация и авторизация

| Команда                                          | Описание                      |
| ------------------------------------------------ | ----------------------------- |
| `/register (пароль) (повтор пароля)`             | Зарегистрироваться на сервере |
| `/login (пароль)`                                | Авторизоваться на сервере     |
| `/changepassword (старый пароль) (новый пароль)` | Сменить пароль                |
|                                                  |                               |

***

### Система скинов

| Команда                          | Описание                                        |
| -------------------------------- | ----------------------------------------------- |
| `/skin set (никнейм)`            | Установить скин по нику лицензионного аккаунта  |
| `/skin url (ссылка на картинку)` | Установить скин по прямой ссылке на изображение |
| `/skin clear`                    | Сбросить текущий серверный скин                 |
| `/skin random`                   | Установить случайный скин                       |

***

### Чаты

| Команда                      | Описание                                                               |
| ---------------------------- | ---------------------------------------------------------------------- |
| `(сообщение)`                | Отправить сообщение в локальный чат (видят игроки в радиусе 80 блоков) |
| `!(сообщение)`               | Отправить сообщение в глобальный чат (видят все игроки сервера)        |
| `/msg (никнейм) (сообщение)` | Отправить личное сообщение указанному игроку                           |
| `/r (сообщение)`             | Ответить на последнее полученное личное сообщение                      |

***
